# FitnessTracker
Detta är en enkel träningsapp som skapades som ett skolprojekt.
Appen låter:

*	söka efter mat och se näringsvärden
*	räkna kalorier och BMI
* få träningsförslag

Data hämtas från gratis API:er ([OpenFoodFacts](https://world.openfoodfacts.org) och [ExerciseDB](https://exercisedb.dev)).
